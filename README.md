Product Input Loop (CLI)

📌 Description

A simple Python program that allows users to enter product data repeatedly, validating inputs and calculating the total cost.

---

🚀 Features

- Input product name
- Validate price (must be a positive number)
- Validate quantity (must be a positive integer)
- Calculate total cost
- Repeat process until user exits

---

▶️ How to Run

python your_file.py

---

🔄 How It Works

1. The program asks for a product name.
2. It validates:
   - Price → must be a positive number
   - Quantity → must be a positive integer
3. Displays a summary:
   - Product
   - Price
   - Quantity
   - Total cost
4. Asks if the user wants to continue.

---

💡 Example Output

Product: Apple
Unit price: 2
Quantity: 5

Product: Apple | Price: 2.0 | Quantity: 5 | Total: 10.0
